#  Calculadora em Java (Console)

Projeto de **calculadora interativa em Java**, executada via console, desenvolvido com o objetivo de praticar conceitos fundamentais da linguagem, como entrada de dados, controle de fluxo e laços de repetição.

---

##  Funcionalidades

- Adição
- Subtração
- Multiplicação
- Divisão (com validação de divisão por zero)
- Cálculo de porcentagem
- Potenciação

O programa permite realizar múltiplos cálculos consecutivos até que o usuário decida encerrar a execução.

---

##  Tecnologias Utilizadas

- Java
- Scanner (entrada de dados via console)
- Estruturas de controle:
  - `if / else`
  - `switch`
  - `do while`

---

##  Como Executar

1. Clone o repositório:
   ```bash
   git clone (https://github.com/LucasFreitasLF/Calculadora-java.git)

2. Acesse o diretório do projeto:
cd calculadora-java

3. Compile o código:
javac Main.java

4. Execute o programa:
java Main

##Regras de Funcionamento

 - O sistema solicita dois números para cada operação.

 - Em caso de divisão por zero, a operação é cancelada e uma mensagem de erro é exibida.

 - A operação de porcentagem considera:

 - `primeiro número como o percentual`

 - `segundo número como o valor base`

  ##Exemplo:
  10% de 200 = 20


## Possíveis Evoluções

Separação das operações em métodos

Tratamento de erros de entrada com try/catch
